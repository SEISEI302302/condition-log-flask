# Condition Log App

Flask を使用したシンプルなコンディション記録アプリです。

## 機能
- 体調（energy）とメンタル（mind）の数値記録
- コメントの追加
- 記録の編集・削除
- SQLite によるデータ保存

## 使用技術
- Python
- Flask
- SQLite
- Git / GitHub

## 起動方法
```bash
python app.py
ブラウザで http://127.0.0.1:5000 にアクセスしてください。
```
